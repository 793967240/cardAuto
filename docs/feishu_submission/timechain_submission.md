## 核心玩法介绍（200 字内）

《时序录》是一款清透新国风仙侠题材的时间轴卡牌构筑 Roguelike。玩家在爬塔过程中收集卡牌、宝石和遗物，将卡牌放入固定底座形成自动循环执行的战斗链条。战斗中玩家与敌人的时间轴同步推进，卡牌按顺序释放，循环结束会进入修整窗口。玩家需要通过短链高频、长链爆发、充能、回响、打断等协同组合优化链条节奏，击败普通敌人、精英与 Boss，完成一局构筑成长。

## 操作指南

### 基础流程

1. 在主菜单选择开始游戏。
2. 选择初始卡组，进入地图。
3. 点击当前可达地图节点，触发战斗、宝箱、营火或 Boss。
4. 战斗胜利后选择奖励，并进入构筑界面调整链条。
5. 重复路线选择与构筑优化，最终挑战 Boss。

### 键鼠操作

| 操作 | 说明 |
|---|---|
| 鼠标左键点击 | 选择菜单、地图节点、奖励与按钮 |
| 鼠标左键拖拽 | 在构筑界面拖拽卡牌入槽、调整卡牌顺序、拖拽宝石入槽 |
| 鼠标右键 | 取消、返回，或将卡牌/宝石快速取出 |
| 鼠标悬停 | 查看卡牌、底座、敌人意图等详情 |
| 空格 | 暂停或继续战斗 |
| 数字键 1 / 2 / 3 | 切换 1x / 2x / 4x 战斗速度 |
| Tab | 打开或关闭时间轴甘特图对比 |
| Esc | 返回或打开设置菜单 |
| Enter | 确认，或进入下一节点 |
| Q | 投降当前战斗 |

### 触屏兼容

- 点击用于选择菜单、地图节点、奖励与按钮。
- 拖拽用于移动卡牌或宝石。
- 长按 0.4 秒可查看卡牌详情，作为鼠标悬停的触屏替代操作。
